# Airbnb Clone

A modern Airbnb clone built with Next.js, TypeScript, and Tailwind CSS.

## Features

* Home page with category filters
* Catalog page with sorting options
* Dynamic room detail pages
* Responsive design
* Reservation simulation
* Reusable React components

## Technologies

* Next.js 16
* React
* TypeScript
* Tailwind CSS

## Pages

### Home Page

Users can explore rooms and filter them by category.

### Catalog Page

Users can view all rooms and sort them by price.

### Room Detail Page

Users can view room information and simulate a reservation.

## Installation

Run the development server:

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```
